Project SiGMA

Project Info:

The objective of the Academic Mobility Management System - SiGMA is to gather in the same software all the necessary information that supports the international academic mobility program. That is, through it, the user can register all institutions and people involved in the programs, crossing the data according to the objective of the desired research. It is a way that the user will find to record and update all the international movement that takes place within the institution. SiGMA will only be fully useful if it is constantly updated and if its information is being expanded as mobility grows.
