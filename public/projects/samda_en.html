Project SAMDA

Project Info:

This project was the result of a thesis work, where an Open Data Maturity Model - MMDA was proposed. Given the complexity of the model proposed in the thesis, it was necessary to develop this system called Open Data Maturity Assessment Support System - SAMDA. Through this system it is possible to apply a dynamic questionnaire, where, according to the answers informed by the user, the questionnaire is shaped. It is also possible to verify through the answers informed the level of maturity of the organization in relation to the good practices of openness and publication of open data.
