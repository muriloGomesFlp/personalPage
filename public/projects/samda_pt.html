Projeto SAMDA

Informações do Projeto:

Este projeto foi fruto de um trabalho de tese, onde foi proposto um O Modelo de Maturidade de Dados Abertos - MMDA. Dado a complexidade do modelo proposto na tese, se fez necessário o desenvolvimento deste sistema denominado Sistema de Apoio a Avaliação da Maturidade de Dados Abertos - SAMDA. Por meio deste sistema é possível aplicar um questionário dinâmico, onde conforme as respostas informadas pelo usuário o questionário vai se moldando. Também é possível verificar através das respostas informadas o nível de maturidade da organização em relação às boas práticas de abertura e publicação de dados abertos.
